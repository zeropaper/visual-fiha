# Playback

The playback can be controlled using the timeline at the bottom of the interface. You can play, pause, and scrub through the timeline to see how your visuals change over time. 

## Known Duration Mode

When audio files are used, the timeline will automatically adjust to match the duration of the longest audio track. This allows you to synchronize your visuals with the music.

## Unknown Duration Mode

If no audio files are used for playback, the timeline will start but will not have a defined end. You can still play and pause and use a microphone input, but the timeline will scale dynamically based on the current time. This allows for more open-ended visualizations that can evolve indefinitely.
